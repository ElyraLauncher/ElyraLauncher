# Security Policy

## Supported Scope

Security review applies to the current `main` branch and active maintenance branches. Reports should distinguish between ROM/system behavior and standalone smoke APK behavior.

## Reporting

No official private security contact is published for this repository yet. Until one exists, do not place sensitive exploit details in public issues. Open a minimal public issue requesting maintainer contact, or use the repository owner's published security contact if one is later added to the project profile.

## Expectations

Security fixes must preserve Launcher3 and Quickstep architecture. Do not hide issues by removing platform integration code, disabling privileged behavior, or replacing ROM validation with standalone APK checks.
